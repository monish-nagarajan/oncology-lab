# 🧬 Cancer Cell Growth Virtual Lab

This project is a virtual biotechnology lab that simulates the growth of normal and cancer cells under different experimental conditions.

## 🔬 Features
- Compare normal vs cancer cell growth
- Adjust nutrient levels
- Apply drug dosage
- Visualize growth using graphs
- Real-time observations

## 🧠 Concept
- Normal cells grow linearly
- Cancer cells grow exponentially
- Drug reduces cell proliferation

## ⚙️ Tech Stack
- Python
- Streamlit
- NumPy
- Matplotlib

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
